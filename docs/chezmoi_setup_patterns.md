Learned chezmoi setup on Daniel Lynch macOS: chezmoi source at `~/.local/share/chezmoi`, config at `~/.config/chezmoi/chezmoi.toml`. Patterns: XDG-compliant zsh via `~/ .zshenv` bootstrapping `ZDOTDIR=$XDG_CONFIG_HOME/zsh`; real configs in `dot_config/zsh/` with templates. Uses 1Password service account mode in chezmoi config (`[onepassword] mode="service"`), and 1Password shell plugin init in `.zshrc` plus SSH agent socket exported in `.zprofile` template. Modular mise config follows atoms/molecules taxonomy under `dot_config/mise/conf.d/{atoms,molecules}` with path SSOT registry in `dot_config/mise/registry/private_paths.toml` and parameter definitions in `docs/SSOT-PARAMETER-DEFINITIONS.toml`. Repo sprawl risks: runtime/generated dirs tracked in source repo (`.cache/`, `.claude-flow/metrics`, `.swarm/memory.db`, `scripts/validators/.venv`, `.uv-cache`, `.pytest_cache`). Chezmoi scripts: `run_once_cleanup_shell.sh` removes rogue `~/.zshrc/.zprofile/...`; `run_onchange_darwin-install-packages.sh.tmpl` installs brew packages list.
